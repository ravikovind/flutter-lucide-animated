Flutter Lucide Animated

CDN for animated Lucide icons in Flutter

Registry

Get list of all available icons

Icon Data

Get animation data for a specific icon

GET /v1/icons/{icon-name}.json

Usage

Add the package to your Flutter project:

flutter pub add flutter_lucide_animated

Add icons using the CLI:

dart run flutter_lucide_animated add flame settings bell
